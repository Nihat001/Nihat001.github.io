# Nihat Masimli Portfolio

Professional one-page portfolio website for Nihat Masimli, a Full Stack Web Developer based in Budapest, Hungary.

Built with React and Vite for GitHub Pages deployment at <https://nihat001.github.io>.

## Tech Stack

- React
- Vite
- CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is generated in the `dist/` directory.
